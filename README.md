# StoryGraph

An AI-powered pick-your-path adventure game built with LangGraph and Ollama.

## Features

- Natural language player input
- AI action interpretation
- Branching story paths
- Local LLM using Ollama
- LangGraph state management

## Tech Stack

- Python
- LangGraph
- Ollama
- Qwen 2.5
- Pydantic
